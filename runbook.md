
# Runbook (Bedienungsanleitung)
![Runbook Abbildung 1](screenshots\to-do Main.png)

## Übersicht der Funktionen in Abbildung 1

1. **Neue Liste erstellen**
2. **Anzeige/Auswahl der vorhandenen Listen**
3. **Checkbox eines To-dos in der ausgewählten Liste**
4. **Informationen des To-dos in der ausgewählten Liste**
5. **Erstellen eines neuen To-dos in der geöffneten Liste**
6. **Öffnen des Filterfensters zum Filtern innerhalb einer Liste**
7. **Filtern über alle Listen anhand des Fälligkeitsdatums (kürzeste Fälligkeit oben)**
8. **Löschen einer Liste**  
   - Voraussetzungen:  
     - mindestens ein To-do vorhanden  
     - alle To-dos abgeschlossen
9. **Name der aktuell ausgewählten Liste**

---

# Eine neue Liste erstellen
![Runbook Abbildung 2](pfad/zu/bild2.png)

Durch Klick auf den Button zum Erstellen einer neuen Liste (Abb. 1, Feld 1) öffnet sich ein Eingabefeld.  
Der Name der neuen Liste wird eingegeben und per Button bestätigt.

➡ Ergebnis: **Die Liste wurde erfolgreich erstellt.**

---

# Ein neues To-do erstellen
![Runbook Abbildung 3](pfad/zu/bild3.png)

Nachdem eine Liste ausgewählt wurde (Abb. 1, Feld 2), wird der Button zum Erstellen eines neuen To-dos gedrückt (Abb. 1, Feld 5).  
Es öffnet sich eine Eingabemaske zur Konfiguration von:

- Titel / Beschreibung  
- Priorität  
- Fälligkeitsdatum  

Die Erstellung erfolgt durch Klick auf **"To-do hinzufügen"**.

---

# Quittieren von abgeschlossenen To-dos & Löschen einer Liste
![Runbook Abbildung 4](pfad/zu/bild4.png)

Nach Auswahl einer Liste werden alle enthaltenen To-dos angezeigt.  
Über die Checkbox kann ein To-do als abgeschlossen markiert werden.

Eine Liste kann gelöscht werden, wenn:

- mindestens **ein To-do vorhanden** ist  
- **alle To-dos abgeschlossen** wurden  

Das Löschen erfolgt über den Button am unteren Rand der Ansicht.

---

# Filtern in Listen
![Runbook Abbildung 5](pfad/zu/bild5.png)

Nach Auswahl der gewünschten Liste wird der Button **„Filtern“** (Abb. 1, Feld 6) gedrückt.  
Daraufhin öffnet sich das Filtermenü (oben rechts sichtbar).

Die gesetzten Filter verändern die ausgegebene Liste in der Mitte der Abbildung.

---

# Filtern nach nächsten Fälligkeiten über alle Listen
![Runbook Abbildung 6](pfad/zu/bild6.png)

Die Funktion wird ausschließlich über den Button unten links aktiviert (Abb. 1, Feld 7).  
Es werden alle Listen nach Fälligkeiten sortiert dargestellt, beginnend mit der kürzesten Deadline.
