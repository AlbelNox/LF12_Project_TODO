# LF12_Project_TODO
Project in School for Projectmanagement 

## Projectgoal
```md
For adding some toDos to the applikation, we will provide an UI with HTML and CSS.
In the backend, will be all the logic implemented. Like `save`, `new ToDo List`, `delete`, `check (done)`, `uncheck (undone)`, ...
The Todos will be saved local in an DB on the client.
```

## How
```md
- ToDo App
   - Frontend (HTML / CSS)
   - Backend (<Programmlanguage>)
   - Local DB for saving
```
